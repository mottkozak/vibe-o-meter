Vibe-o-meter

Repository: https://github.com/mottkozak/vibe-o-meter

A satirical personality quiz inspired by Myers-Briggs, fantasy archetypes, and a healthy disrespect for pseudoscience.

The quiz presents users with 30 questions across 5 archetype compasses, calculates their alignment on multiple personality axes, and generates a 6-letter type code + archetype title along with a humorous write-up.

The site is built using:
	•	Vite
	•	React
	•	TypeScript
	•	Static JSON data files
	•	GitHub Pages hosting

The app is 100% static and runs entirely in the browser.

No backend. No database. No tracking. No pseudo-scientific authority.

⸻

Running locally
npm install
npm run dev

Building
npm run build

Deployment

Deployment is automatic via GitHub Actions → GitHub Pages.

The site will be available at:
https://mottkozak.github.io/vibe-o-meter/

Project philosophy

This project intentionally pokes fun at personality typing systems while still creating something:
	•	fun
	•	visually satisfying
	•	weirdly insightful
	•	highly shareable
