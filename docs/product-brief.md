Product Brief

Project Goal

Create a lighthearted personality quiz that feels like:
	•	a fantasy character creator
	•	a parody of Myers-Briggs
	•	an interactive archetype map

Users answer questions and receive:
	•	a 6-letter type code
	•	an archetype title
	•	a five-compass personality profile
	•	a humorous write-up
	•	celebrity “vibe matches”

⸻

Tone

The tone should be:
	•	playful
	•	irreverent
	•	self-aware
	•	slightly absurd
	•	but still flattering

The site should explicitly acknowledge that the quiz is not scientific.

⸻

Target Experience

The quiz should feel like:
	•	an RPG stat sheet
	•	a personality alignment chart
	•	a fantasy archetype generator

⸻

User Flow

Landing Page
→ Quiz Start

Quiz (5 sections)
	1.	Power Compass
	2.	Order Compass
	3.	Discipline Compass
	4.	Social Style Compass
	5.	Risk Compass

Each section has 6 questions.

Total questions: 30

After completion:

Results Page shows:
	•	archetype title
	•	personality code
	•	strengths
	•	pitfalls
	•	celebrity matches
	•	5 compass charts

⸻

Constraints

The application must:
	•	be fully static
	•	run entirely client-side
	•	load quiz data from JSON
	•	require zero paid services
	•	deploy via GitHub Pages



## Pitch
A playful, satire-forward personality quiz that mocks MBTI’s pseudo-science energy while delivering a surprisingly satisfying “character class” result using fantasy archetype compasses.

## Goals
- Be fun and shareable (social-friendly results page)
- Be obviously non-scientific (explicit disclaimer + jokes)
- Still feel coherent and “accurate enough” (consistent scoring, visual charts)
- Produce:
  1) A 6-letter “BS type code” (64 outcomes)
  2) A 5-compass profile (20 quadrant labels total)
  3) A comedic write-up + celeb/historical “vibe matches”

## Audience
- People who like MBTI, astrology, Hogwarts houses, D&D alignments
- Teams wanting a fun icebreaker (explicitly not HR/clinical)

## User Flow (MVP)
Landing → Disclaimer → 5 sections (6 Q each) → Results → Share

## Tone Rules
- Friendly, winky skepticism (“peer-reviewed by a raccoon” energy)
- Never medical/diagnostic
- Never mean or insulting
- Results should be flattering-ish but honest about pitfalls

## MVP Deliverables
- Quiz UI with progress
- Scoring engine
- Results page with:
  - 6-letter BS code + 64 title
  - 5 mini compass charts (dots on grid)
  - Write-up
  - Famous “vibes” list
  - Share card (optional MVP+)
